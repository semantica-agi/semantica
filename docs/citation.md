## BibTeX

```bibtex
@software{semantica2026,
  title   = {Semantica: An Open Source Framework for Semantic Layers and Knowledge Engineering},
  author  = {Hawksight AI},
  year    = {2026},
  url     = {https://github.com/semantica-agi/semantica},
  version = {0.5.0},
  doi     = {10.5281/zenodo.XXXXXXX}
}
```
